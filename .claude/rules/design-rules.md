# Design Rules

## Design Tokens
| Token | Hex | Use |
|-------|-----|-----|
| Dragon Red | `#C1121F` | Primary CTA, action, authority |
| Deep Crimson | `#8E0F19` | Hover states, depth |
| Carbon Black | `#111111` | Primary background |
| Graphite | `#1C1C1C` | Cards, panels, sections |
| Brushed Steel | `#BFC3C7` | Headlines, iconography, lines |
| Alloy Grey | `#8D9398` | Supporting UI tones |
| Clean White | `#F8F8F8` | Body text, readable contrast |
| Welsh Green | `#00843D` | National identity only — use sparingly |

## Aesthetic
Dark luxury automotive. Not loud. Not cheap. Not "boy racer". Not a generic trade site.
Large breathing space. Strong grid. Elegant spacing. Premium contrast.
Reference: Polestar × Apple × Linear.

## Typography
- Headlines: Space Grotesk (modern, engineered, assertive)
- Body: Inter (trustworthy, contemporary, readable)
- Buttons: bold uppercase, letter-spacing -0.03em to -0.05em

## Layout Rules
- Corner radii: 4–8px max — squared/luxe aesthetic, no pill buttons ever
- Scroll animations: subtle reveal via Intersection Observer
- Mobile-first. Sticky mobile CTA bar on small screens.
- Tap targets ≥44px

## Brand Voice
- Calm authority. Professional confidence. No fluff.
- Sound like: "We'll handle it." / "Booked. Confirmed. Sorted." / "Professional service at your door."
- Never: "Cheap deals available!!!" / "Best prices guaranteed mate" / "Give us a bell"

## Approved CTA Copy Bank
- Book Your Vehicle
- Get Fast Quote
- Call Local Experts
- Request Same-Day Help
- Request Booking

## Never Use
- Discount or urgency language ("limited time", "best prices", "guaranteed cheapest")
- Greasy garage / cluttered workshop imagery
- Generic trade website patterns
- Pill-shaped buttons
- Welsh Green (`#00843D`) except for national identity moments
- "Priority" framing on the booking form (removed in Session 3)

## Conversion Psychology
- Justify price through: convenience, speed, professionalism, trust — never parts cost
- Risk reversal: "Transparent estimates before work begins"
- Friction removal: tap-to-call, WhatsApp CTA, short form
- Trust mechanics: real reviews with names + towns, technician credentials, clean invoices
